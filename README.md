# My First AI

Une intelligence artificielle conversationnelle développée en JavaScript.

## Description

My First AI est un projet personnel conçu pour apprendre les bases de l'intelligence artificielle, du traitement du langage naturel et du développement logiciel. L'objectif est de créer un assistant capable de comprendre les messages des utilisateurs, générer des réponses pertinentes et mémoriser les conversations.

## Fonctionnalités

* Réponses automatiques aux messages
* Détection de mots-clés
* Gestion des conversations
* Système de mémoire utilisateur
* Sauvegarde de l'historique des discussions
* Architecture évolutive pour ajouter de nouvelles fonctionnalités

## Technologies utilisées

* JavaScript
* Node.js
* JSON
* Git & GitHub

## Installation

```bash
git clone https://github.com/votre-utilisateur/My-first-Al.git
cd My-first-Al
npm install
npm start
```

## Objectifs du projet

* Comprendre le fonctionnement d'une IA conversationnelle
* Apprendre à structurer un projet JavaScript
* Expérimenter différents systèmes de mémoire et de raisonnement
* Développer progressivement un assistant plus intelligent
